# WebMakr Builder To-Do List
*Essential features required for a complete, professional website builder.*

## 1. Core Pages & Legal Compliance
- [ ] **Privacy Policy Generator**: Auto-generate based on site settings (Analytics, Affiliates).
- [ ] **Terms of Service**: Standard boilerplate for content sites.
- [ ] **EULA**: Required if distributing software (like the "GetDiskSpace" app).
- [ ] **Affiliate Disclosure**: Global, customizable disclosure text (Footer/Top of Post).
- [ ] **Cookie Policy & Banner**: GDPR/CCPA compliance banner.
- [ ] **Contact Page**: Built-in contact form or localized 'mailto' links.
- [ ] **404 Error Page**: Custom "Page Not Found" with helpful navigation links.

## 2. SEO & Metadata
- [ ] **Global Meta Tags**: Description, Keywords, Author, Charset.
- [ ] **Open Graph (OG) Tags**: Default images/titles for social sharing (Facebook, LinkedIn).
- [ ] **Twitter Cards**: Large summary cards for Twitter/X sharing.
- [ ] **Sitemap.xml**: Auto-generation including all Articles and Static Pages.
- [ ] **Robots.txt**: Rules for search engine crawlers.
- [ ] **Canonical URLs**: Handling for duplicate content/syndication.

## 3. Integrations & Analytics
- [ ] **Google Analytics 4 (GA4)**: Easy ID input (G-XXXXXXXX).
- [ ] **Plausible / Simple Analytics**: Privacy-friendly alternatives.
- [ ] **Social Media Links**: Centralized management for Footer/Header icons (X, TikTok, LinkedIn, YouTube, GitHub).
- [ ] **Email Capture**: Integration with ConvertKit, Mailchimp, or generic newsletter forms.

## 4. Performance & Assets
- [ ] **Favicon Generation**: Auto-generate `.ico`, `.png`, and `apple-touch-icon` from one source image.
- [ ] **Asset Optimization**: WebP conversion for images.
- [ ] **Code Minification**: Minify HTML/CSS/JS for production builds.

## 5. Security & Technical
- [ ] **SSL Enforce**: (If hosting managed) - usually handled by Netlify/Vercel.
- [ ] **Security Headers**: Content-Security-Policy (CSP) headers.
